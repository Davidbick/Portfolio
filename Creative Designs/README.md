# Creative Designs

This folder contains creative and visual design work produced for events, organizations, and personal projects. These materials focus on clear communication, visual engagement, and community outreach. Many of the designs were created to support student organizations, residential life programming, and digital media initiatives.

---

## Folder Contents

### AI Designs
A collection of graphics created with the assistance of AI image generation tools. These designs explore the use of generative tools for creating event flyers, clothing, and digital artwork.

### Resident Assistant
Event boards and promotional materials created while serving as a Resident Assistant. These designs were used to promote events, communicate important information to residents, and enhance community engagement within the residence hall.
