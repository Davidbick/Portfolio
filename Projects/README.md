# Projects

This folder contains engineering and technical projects involving embedded systems, AI, IoT, and hardware-software integration. These projects demonstrate the process of designing, building, and testing systems that combine electronics, software, and communication protocols to solve real-world problems.

Each project folder includes a dedicated README describing the motivation, system architecture, technologies used, and key outcomes.

## Folder Contents

### EARTH Automated Recycling
A project developed during an IEEE hackathon focused on improving recycling accessibility and efficiency. The system explored automation concepts to assist users in properly sorting recyclable materials.

---

### Safe&Sound
An embedded AI wearable system designed to classify environmental sounds in real time. The device processes audio data locally to identify important sounds in the environment. The project explores embedded machine learning, sensor data processing, and real-time inference on resource-constrained hardware.

---

### Local-First Smart Home Hub
A smart home hub designed around a local-first architecture, prioritizing reliability and privacy.

The system integrates Matter over Thread for wireless IoT devices and Power Line Communication (PLC) for mains-powered devices. By supporting both communication methods, the hub enables robust connectivity across battery-powered and wall-powered smart home devices while maintaining local control. This design allows for plug-and-play device integration, simplifying setup and improving ease of use for end users.
