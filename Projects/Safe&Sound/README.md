# Safe&Sound 

This folder contains the **project documentation and assets** for *Safe&Sound*, an intelligent wearable system designed to enhance safety and awareness for individuals with hearing impairments.

Safe&Sound combines embedded systems, edge AI, and IoT to detect environmental sounds in real time and provide immediate haptic and visual feedback to the user.

## Project Summary  

Safe&Sound is designed to address the lack of accessible, real-time sound awareness for individuals with hearing impairments. The system uses on-device machine learning to classify sounds such as alarms, doorbells, and alerts, and communicates them through a wearable interface.

Key aspects of the system include:  
- Edge AI for real-time sound classification  
- Low-latency wearable feedback (vibration + LED)  
- Bluetooth Low Energy (BLE) communication  
- Cross-platform mobile application for customization and monitoring  
- Secure and scalable IoT architecture
